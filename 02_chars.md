# Символы

#### Пример 1
Получение кода по символу и обратно.
```c
char a = 'a';
int code = (char) a;
char symbol = (int) code;
if (a == symbol) {
    printf("a == symbol\n");
} else {
    printf("a != symbol\n");
}
```

#### Пример 2
Ввод и вывод.
```c
char a;
scanf("%c", &a);
printf("%c", a);
```

#### Пример 3
Сравнение символов.
```c
char a = '0', b = '9';
if (a < b) {
    printf("%c < %c\n", a, b);
} else {
    printf("%c >= %c\n", a, b);
}
```

#### Пример 4
Игнорирование пробелов.
```c
char a;
do {
    scanf("%c", &a);
    if (a != ' ') {
        printf("%c", a);
    }
} while (a != '\n');
```
